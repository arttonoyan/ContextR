# ContextR.Hosting.AspNetCore

ASP.NET Core ingress integration for ContextR.

Adds startup-filter based middleware to extract incoming headers into typed context and supports ingress enforcement options.

## Install

```bash
dotnet add package ContextR.Hosting.AspNetCore
```
