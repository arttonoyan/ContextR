# ContextR.Propagation.InlineJson

Inline JSON payload strategy for complex mapped properties.

Provides JSON serialization/deserialization behavior for mapped values and supports oversize strategy defaults.

## Install

```bash
dotnet add package ContextR.Propagation.InlineJson
```
