# ContextR.Propagation.Mapping

Fluent mapping DSL and default mapping-based propagator.

Use this package to map typed context properties to carrier keys (headers/metadata) using `MapProperty(...)` or `Map(...)`.

## Install

```bash
dotnet add package ContextR.Propagation.Mapping
```
