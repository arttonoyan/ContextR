# ContextR.Propagation.Token

Token/reference payload contracts for out-of-band propagation.

Use this package to define token store/codec abstractions when payloads should be referenced externally (for example Redis or database providers).

## Install

```bash
dotnet add package ContextR.Propagation.Token
```
