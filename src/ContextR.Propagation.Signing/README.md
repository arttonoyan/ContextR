# ContextR.Propagation.Signing

HMAC-based context signing for ContextR propagation.

Provides tamper detection for context headers across service boundaries. Signs all mapped property headers on inject and verifies them on extract using HMAC-SHA256.

## Install

```bash
dotnet add package ContextR.Propagation.Signing
```
